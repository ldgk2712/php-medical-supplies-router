# PHP Medical Supplies Router

Simple demo project with a minimal router and views for managing supplies.

Structure:

- public/ - front controller and assets
- src/ - PHP source (Core, Controllers, Support, Data)
- views/ - simple PHP views

Run:

1. Install autoloader: `composer dump-autoload`
2. Serve `public` directory (e.g., `php -S localhost:8000 -t public`)
